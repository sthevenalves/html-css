Usando Links Externos

Hyperlinks permitem que você ligue um ponto ao outro na world wide web

Para criar Hyperlinks, devemos criar âncoras através da tag A. O principal atributo dessa tag é o href, que cria uma referencia hipertexto.

Usando links internos

Por padrão, sempre que um vistante clica em um Hyperlink, o site do destino abre na mesma janela, ou seja, o conteúdo anterior deixa de ser exibido para mostrar o novo.

Para controlar isso, existe outro propriedade da tag A, que é o target, que suporta os seguintes valores:

  • Blank (abre em uma nova janela)
  • Self (abre na janela ou frame atual)

Para link interno, ou link local,

Esse link é seu ou de outros?

Existe un recurso bem interessante para indicar qual é a natureza do destino usando o atributo rel, ele aceita os seguintes valores:

  • Next (indica que o link é para próxima parte do doc atual)
  • Prev (indica que o link é para parte anterior do doc atual)
  • Author (indica que o link é para o site do autor do artigo)
  • External (indica que o link é para outro site que não parte do site)

Links para Download

Para add links para download, apos o comando A, no href da um ctrl + space para buscar o arquivo, depois utilizar o comando download = "nome do arquivo" e o type para indicar ao navegador que tipo de arquivo esta sendo baixado, alguns media types bem usados:

  • application/zip
  • video/mp4
  • audio/mpeg
  • text/html
  • video/h2g4
  • font/ttf
  • text/css
  • video/jpeg
  • image/jpeg
  • text/javascript
  • audio/aac
  • image/png
  • Baixar o livro em PDF
  • Baixar livro em ZIP

OBS: o arquivo para download deve ta dentro da pasta do site.
