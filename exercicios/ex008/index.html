Listas ordenadas (OL)

Ordered list(OL) é uma tag para ordernar listas é usado juntamente de seu delimitar (LI)

  1. Acordar
  2. Higiene Matinal
  3. Aeróbico
  4. Tomar banho
  5. Café da Manhã

O /LI no final é opcional, ele funciona so com a entrada LI

  1. Batata
  2. Chocotone
  3. Refrigerante
  4. Queijo

O OL possui um parametro type, onde configuramos o marcador, que são 1,A,a,I,i.
Também pode indicar o início da contagem, usando o parametro start

Listas não ordenadas (UL)

Também chamadas de marcadores, são aquelas onde a ordem não influenciará no significado da lista.
Ela é apenas uma ótica maneira para organizar os itens

  • Papel Higienico
  • Tenis
  • Calças
  • Frutas

Assim como o OL, o UL também possui um parametro type, sendo eles:
disc - padrão(bolinha preta, circle - uma bola com uma borda preta e sem preenchimento, square - um pequeno quadrado

Listas de Definições

Como se fosse no dicionário, temos termos e definições. É uma lista sem marcadores
Toda lista de definições esta dentro de uma tag DL(definition list), cada termo é um DT(definition term) e cada descrição é um DD(definition description)
